Features

⚡

Instant extraction

Detects IPs, domains, URLs, MD5/SHA1/SHA256 hashes, emails, and CVEs automatically as you type. Duplicate IOCs show a ×N badge.

🔄

Defang & refang

Switch between defang mode (safe to share) and refang mode (restore original indicators) with one click.

🔗

Pivot links

Hover any IOC to reveal one-click investigation links: VirusTotal, Shodan, AbuseIPDB, URLhaus, MalwareBazaar, NVD, EPSS.

🛡

YARA rule generator

One click generates a ready-to-use YARA rule from extracted IOCs — strings for IPs, domains, URLs; hash-module conditions for hashes.

⬇

Export anywhere

Download as .txt, .csv, .json, .yar, or copy as a Markdown table for tickets and reports.

📂

Bulk & drag & drop

Drop a file directly on the input zone, or upload multiple .txt / .log files at once — processed and merged instantly.

🧩

Browser extension

Select text on any page → right-click → "Defang IOCs" or "Refang IOCs". Processes in place and copies to clipboard. Works on every site, no tab switching.

🕐

Session history

Every session is auto-saved locally. Restore any past analysis with one click — up to 25 sessions stored privately in your browser.

⚙️

URL param / API mode

Pass ?input=…&format=json to auto-process and get a JSON response — useful for bookmarklets, scripts, and integrations.

Browser extension

Defang & refang without leaving the page

Select any IOC on a threat report, a vendor advisory, or a Slack message — right-click and hit 🔬 Defang IOCs or 🔁 Refang IOCs. The result is copied to your clipboard instantly, with a confirmation toast on the page.

Installation (Chrome / Edge)

1. Download the extension/ folder from the GitHub repo
2. Open chrome://extensions and enable Developer mode
3. Click Load unpacked and select the folder
4. Select text → right-click → 🔬 Defang or 🔁 Refang
